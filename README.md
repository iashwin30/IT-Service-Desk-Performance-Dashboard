# IT Service Desk Performance Dashboard

## 📊 Project Overview
Comprehensive Power BI dashboard analyzing 16,721 IT service desk tickets to identify SLA compliance gaps, CSAT issues, and agent performance bottlenecks.

**Analysis Period:** April – August 2024

## 🎯 Key Findings

### SLA Compliance
- **Current:** 76.8% (Target: 80%)
- **Gap:** -3.2%
- **Insight:** General Inquiry & Access Request categories causing 47.3% of breaches

### Customer Satisfaction (CSAT)
- **Current:** 40%
- **Industry Average:** 70%
- **Gap:** -30% (Critical issue)

### Agent Performance
- **Best Performer:** 83.9% SLA compliance
- **Lowest Performer:** 79.8% SLA compliance
- **Variance:** 4.1% (opportunity for standardization)

## 📈 Dashboard Contents

### Page 1: Overview
- KPI metrics (SLA, CSAT, Avg Resolution Time)
- Monthly trend vs 80% target
- Breach rate by category
- SLA compliance gauge chart

### Page 2: Deep Dive Analysis
- SLA performance by category (conditional formatting)
- Agent performance heat map (Agent 1-80, anonymized)
- 3 key insights with findings
- Priority 1-3 recommended actions

## 💡 Business Recommendations

**Priority 1 - CRITICAL (47.3% impact):**
- Optimize ticket triage for General Inquiry & Access Request
- Target: 76.8% → 80% SLA compliance

**Priority 2 - HIGH:**
- Implement agent coaching for underperformers
- Target: Standardize performance (79.8% → 81%+)

**Priority 3 - MEDIUM:**
- Improve customer communication loops
- Target: Increase CSAT from 40% → 60%+
